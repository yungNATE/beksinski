## About The Project

This project is all about giving credits to the master that is Zdzislaw Beksinski. I made this project in groupe during my Master in UX Design at the <a href="https://www.utc.fr/formations/diplome-de-master/mention-humanites-et-industries-creatives-hic/">University of Compiègne (UTC)</a> in a class called "Interactive and multimedia writing", directed by <a href="https://www.utc.fr/~bouchard/wordpress/">Serge Bouchardon</a> (SI28).
<blockquote>Interactive stories, video games, digital art... To what extent can the characteristics of digital and multimedia media generate new forms of writing? Consideration of the concepts of scriptwriting and interactivity will clarify the issues at stake, particularly from a literary and artistic point of view. You will familiarise yourself with certain software and languages: free software for creating interactive stories (Twine), 3D game engine (Unity), HTML5/CSS3 and JavaScript, image, sound and video processing software. You will design and produce an interactive multimedia project, taking advantage of the specific features of different media and devices (PC, smartphone, tablet, etc.). - Serge Bouchardon</blockquote>
Through this project, you'll rediscover besinski painting merged with desktop interactions (mainly mouse ones though).  

My personnal goal was to improve my skills in creative coding, as I never had any project in this field.
